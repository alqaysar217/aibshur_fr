# **App Name**: أبشر

## Core Features:

- Secure User Authentication: Implement robust phone OTP-based registration and login for all user roles (customer, driver, admin) using Firebase Authentication.
- Product & Store Discovery: Customers can intuitively browse through available stores, explore product categories, and view detailed listings for individual products within the application.
- AI-Powered Smart Search & Recommendations: An AI tool that suggests relevant products and stores based on customer preferences, browsing history, or real-time contextual information to enhance item discovery.
- Shopping Cart & Order Creation: Customers can easily add items to their shopping cart, manage quantities, and proceed to place delivery orders with accurate address selection, all powered by Firestore for persistence.
- Order Workflow & Real-time Status Updates: Customers can track their orders through various stages, and assigned drivers can update the real-time status of deliveries (e.g., accepted, preparing, en route, delivered), with data synchronized via Firestore.
- Driver Order Acceptance & Management: Drivers receive notifications for new delivery requests, allowing them to view order details and accept them for fulfillment, seamlessly updating their status in Firestore.
- Admin Order Oversight Dashboard: An administrative interface for managers to view, filter, and oversee all active, pending, and completed orders, ensuring smooth operation of the delivery platform via Firestore.

## Style Guidelines:

- Primary brand color: A deep, reassuring blue (#2966A3), chosen to convey reliability and trust in a service-oriented application. This hue suggests competence without being overly formal, balancing modern appeal with professionalism.
- Background color: A very subtle, cool-toned off-white (#F0F2F5), providing a clean and open canvas that ensures excellent readability and minimizes eye strain, especially for extended use on mobile devices.
- Accent color: A vibrant aqua-cyan (#1AE6E6) used sparingly to highlight critical call-to-action elements, interactive components, and real-time status indicators. Its vivid contrast with the primary and background colors guides user attention effectively.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly readable, and versatile characteristics across various screen sizes and information densities, providing a clean and objective tone suitable for a functional application.
- Utilize a consistent set of filled or slightly rounded line icons with a clear, contemporary aesthetic, prioritizing instant recognition and a friendly, approachable feel for mobile users.
- Employ a mobile-first, highly responsive grid-based layout. Information is organized into intuitive, accessible cards and clear sections to facilitate one-handed navigation and efficient content scanning, mirroring native mobile application patterns.
- Implement subtle, fast, and smooth micro-animations for state changes, navigation transitions, and feedback on user interactions. These animations should enhance the perception of responsiveness and add a polished feel without causing delays.
